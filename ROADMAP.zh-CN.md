# CueMode 项目路线图

## 🎯 项目愿景

CueMode 致力于成为最专业的 VS Code 提词器扩展，为内容创作者、教育工作者和开发者提供流畅的演示体验。

## ✅ 已完成功能

### 🚀 CueMode 2.0.0 (2025-07-12)

#### 核心新功能
- **🎨 T 键主题循环** - 演示期间即时切换主题
- **⚡ 增强命令系统** - 新增 `changeTheme` 和 `removeLeadingSpaces` 命令
- **📱 智能通知** - 非干扰性状态栏消息（2-3秒自动消失）
- **🌐 完整国际化** - 全面的中英文支持
- **🛠️ TypeScript 架构** - 完整代码库重写，提高可维护性
- **🧪 全面测试** - 71 个测试用例确保可靠性

#### 高级快捷键
- `Ctrl+Shift+T` (Mac: Cmd+Shift+T) - 打开主题选择菜单
- `Ctrl+Shift+R` (Mac: Cmd+Shift+R) - 移除前导空格
- `T` 键 - 提词器模式中即时主题循环

#### 专业提词器功能
- **自动滚屏控制** - 空格键开始/停止
- **实时速度调整** - +/- 键
- **方向切换** - R 键反向滚动
- **7 个专业主题** - 针对不同环境优化
- **极简界面** - 无干扰阅读体验
