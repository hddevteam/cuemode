专业提词器功能

专业视频录制和内容创作所需的一切功能

大字体，易阅读

为专业提词器可读性自动优化字体大小和间距，适用于视频录制和直播。

居中提词器布局

专业提词器布局，具有居中文本和可调整的内容宽度，录制期间获得最佳阅读体验。

7 种精美主题

从精心制作的颜色主题中选择，包括经典、午夜蓝、日落、森林、海洋、玫瑰和反色模式。

自动滚动提词器

专业的自动滚动功能，可定制速度控制（空格键切换，+/- 调节速度，R 键切换方向）。

完全可自定义

调整字体大小、行高、内容宽度、内边距、滚动速度和起始位置，以匹配您的演示需求。

无缝控制

直观的键盘控制：空格键切换自动滚动，+/- 调节速度，R 切换方向，箭头键手动导航，Escape 退出。

实时更新

无需重新进入 CueMode 即可立即看到文本更改，设置更改立即生效。

多语言支持

完全国际化，支持中文和英文界面，根据 VS Code 语言环境自动检测语言。

便捷访问

右键上下文菜单集成和命令面板访问，可从 VS Code 中的任何位置快速激活。

如何使用 CueMode

多种方式将您的代码转换为提词器模式

右键上下文菜单

选择文本或打开任何文件，右键打开上下文菜单， 选择"Cue Mode"进入提词器模式。

命令面板

按 Ctrl+Shift+P (Windows/Linux) 或 Cmd+Shift+P (macOS) 并输入"CueMode: Toggle Prompter Mode"。

键盘导航

使用箭头键、上下翻页、Home/End 键导航， 按 Escape 键退出提词器模式。
