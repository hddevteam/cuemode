为演讲与技术分享而生

全方位重塑您的代码演示体验：从编写到路演，一切浑然天成。

多页幻灯片路演

将您的 Markdown 文档瞬间转化为优雅的多页幻灯片。基于分隔符 `---` 轻松分页，在 VS Code 内部通过左右方向键实现无缝翻页演说。

原生代码格式渲染

精准保留代码片段的原始结构与真实换行效果，无需为了提词器妥协排版，带来原汁原味的技术教程展示体验。

专注模式与自动滚动

启用智能高亮区域并自定义滚动速率，让阅读节奏保持自然流畅，不论是直播还是长篇演讲都不再迷失上下文。

原生多国语言支持

深度适配中、英、德、法、日、韩六国语言的原生界面，精准契合跨国开源项目开发团队和创作者的使用习惯。

极简专业色彩主题

全系重构为沉稳克制的极简风格，7 款精心设计的 WCAG 标准高级主题，保障拍摄画面清爽且消除视觉疲劳与频闪。

极速快捷键控制流

一切核心操作皆由快捷键顺畅驱动：空格键切换自动滚动、方向键控制多镜翻页、F 键聚焦、T 键切换主题，尽在掌控。

如何使用 CueMode

多种方式将您的代码转换为提词器模式

右键上下文菜单

选择文本或打开任何文件，右键打开上下文菜单， 选择"Cue Mode"进入提词器模式。

命令面板

按 Ctrl+Shift+P (Windows/Linux) 或 Cmd+Shift+P (macOS) 并输入"CueMode: Toggle Prompter Mode"。

键盘导航

使用箭头键、上下翻页、Home/End 键导航， 按 Escape 键退出提词器模式。
